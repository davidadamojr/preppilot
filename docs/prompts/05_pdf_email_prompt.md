## 05_PDF_EMAIL_PROMPT.md

**Goal:** Implement adaptive PDF and email updates.

**Prompt:**

Add export and notification features to PrepPilot.

### Features
- Generate PDF summaries via `reportlab`:
  - Weekly overview with prep timeline.
  - Ingredient list grouped by reuse.
  - Adaptive highlights (e.g., “Simplified Thursday plan”).
- Email users adaptive summaries when changes occur:
  - Subject: “Your new prep plan is ready.”
  - Include adaptation rationale and link to app.

### Tone
Encouraging, practical, and emotionally intelligent.
Example: *“You missed last night’s prep — no stress. We’ve adjusted your plan so nothing goes to waste.”*
